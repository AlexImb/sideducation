# Sideducation

> An alternative adaptive learning experience platform (LXP)

## Development

### Setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
